# 🐍 PiXel-Py 🎮  

A simple, pixel-style Snake game built with **HTML, CSS & JavaScript**.  

## 🎮 How to Play  
🕹️ **Press any arrow key** to start.  
⬆️⬇️⬅️➡️ **Move the snake**  
🍏 **Eat food to grow**  
🚧 **Avoid walls & yourself!**  

## 🛠️ Tech Stack  
🔹 **HTML** – Structure  
🔹 **CSS** – Styling  
🔹 **JavaScript** – Game logic
